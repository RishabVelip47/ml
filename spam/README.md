# 📧 Spam Classifier (Naive Bayes)

This is my **first Machine Learning project**, where I built a simple **Spam Classifier** using the **Naive Bayes algorithm**. The main goal was to learn the basics of ML and understand how text classification works.

---

## 🚀 Project Overview

This project classifies messages into:

* **Spam** ❌
* **Ham (Not Spam)** ✅

It uses basic **Natural Language Processing (NLP)** and **Machine Learning** techniques.

---

## 🧠 Algorithm Used

* **Naive Bayes (MultinomialNB)**

  * Simple and effective for text classification
  * Based on probability
  * Works well with Bag of Words

---

## 🛠️ Technologies

* Python
* NumPy
* Pandas
* Scikit-learn
* NLTK

---

## 🔄 Workflow

1. Data Cleaning
2. Text Preprocessing
3. Feature Extraction (Bag of Words)
4. Model Training
5. Evaluation

---

## 📂 Project Structure

```bash
spam/
│── model.pkl              # Trained Naive Bayes model
│── vectorizer.pkl         # Text vectorizer (BoW)
│── spam_classifier.py     # Python script for training/prediction
│── spam-classifier.ipynb  # Jupyter notebook (experiments & learning)
```
